Questionários
Questão 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

Resposta: A função de um servidor web é como a de um atendente de loja na internet. Quando você digita um endereço de site, ele atende seu pedido, pega as páginas do site e manda para o seu computador para você ver. Um exemplo famoso é o Apache.
Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

Resposta: A principal coisa do NGINX é que ele é muito bom em lidar com muitos pedidos ao mesmo tempo, sem ficar lento. Ele é usado em sites que têm muita gente acessando porque consegue entregar as coisas rápido para todo mundo.
Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

Resposta: HTTP: É como mandar uma carta sem envelope. Qualquer um pode ler o que está escrito. HTTPS: É como mandar uma carta com um envelope bem lacrado. Só quem recebe pode ler. O HTTPS é importante hoje em dia porque ele protege seus dados (senhas, cartão de crédito), faz o site parecer mais confiável (com o cadeadinho na barra de endereço) e até ajuda o site a aparecer melhor no Google.
Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

Resposta: As portas em um computador são como os portões de um prédio. Cada portão leva a um tipo de serviço diferente. A porta 443 é muito importante porque é por onde passam todas as informações seguras na internet, tipo quando você acessa seu banco ou faz uma compra online.
Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

Resposta: Um endereço IP é como o CEP da sua casa na internet. Cada aparelho (celular, computador) tem um número IP único para ser encontrado. Ele é importante porque é como os computadores se acham para mandar e receber informações na rede.
Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

Resposta: Hospedagem VPS é como morar em um apartamento em um prédio. Você divide o prédio (o servidor grande) com outros, mas tem seu próprio apartamento (seu pedaço do servidor) com seus recursos garantidos. É boa para sites que estão crescendo, que já têm bastante visita ou que precisam de mais controle do que uma hospedagem mais simples.
Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

Resposta: O Caddy é um servidor web mais moderno e fácil de usar. As vantagens dele são: Ele coloca o cadeadinho de segurança (HTTPS) sozinho, sem você precisar fazer nada. É mais simples de configurar. Já vem com as tecnologias mais rápidas para sites.
Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

Resposta: Para programas feitos em .NET (C#), o melhor servidor é o Microsoft IIS. Ele é da própria Microsoft e foi feito para funcionar perfeitamente com os programas .NET, facilitando a vida de quem programa nessa área.
Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

Resposta: No modelo Cliente-Servidor, seu computador (o cliente) pede algo (tipo, uma página de site) para um computador maior (o servidor). O servidor recebe o pedido, prepara o que você quer e manda de volta para o seu computador, que mostra pra você.
Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

Resposta: O Apache Tomcat é um programa que ajuda a rodar sites feitos em Java. Ele não só mostra páginas, mas também faz os programas Java funcionarem. É ótimo para sites mais complexos e grandes, feitos em Java.
Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

Resposta: Um servidor web (como Apache) é como um "entregador de arquivos". Ele manda as páginas prontas (textos, imagens) para o seu navegador. Um servidor de aplicação (como Tomcat) é como um "cozinheiro". Ele pega os ingredientes (dados), cozinha (processa) e monta a página na hora para você. Ele faz as coisas dinâmicas acontecerem.
Questão 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.

Resposta: Um domínio é o nome de um site que a gente lembra fácil, tipo google.com. Ele existe porque é mais fácil lembrar um nome do que um monte de números. A relação com o endereço IP é que o domínio é como o "apelido" do IP. Quando você digita o domínio, um sistema especial (o DNS) traduz esse apelido para o número IP, e assim seu computador encontra o site certo.
Questão 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?

Resposta: A hospedagem em nuvem é como alugar um espaço em um prédio gigante e moderno. As vantagens são: Flexibilidade: Se seu site crescer muito, é fácil aumentar o espaço. Sempre no ar: É mais difícil de sair do ar, porque tem cópias em vários lugares. Menos trabalho: Você não precisa se preocupar em cuidar dos equipamentos. Paga pelo que usa: Você só paga pelo espaço e pelos recursos que realmente usar.
Questão 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.

Resposta: 1. Apache Tomcat: Para programas feitos em Java. 2. Gunicorn: Para programas feitos em Python.
Questão 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?

Resposta: Usar HTTPS hoje em dia é super importante porque é o padrão de segurança da internet. Se um site usa só HTTP, os riscos são: Qualquer um pode ver o que você digita (senhas, números de cartão). Alguém pode mudar as informações que você está vendo. Seu navegador vai avisar que o site não é seguro. O Google vai dar menos importância para o site nas buscas.
Questão 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.

Resposta: Se você tem um blog que de repente ficou muito famoso e vive recebendo milhares de visitas, a hospedagem VPS seria melhor. Na hospedagem compartilhada, seu site ia ficar lento e cair. Com o VPS, você tem mais "espaço" e "poder" só para você, e seu blog aguenta o tráfego sem problemas.
Questão 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.

Resposta: IP: É o "endereço" do computador na rede. Ele serve para achar o computador certo. Porta: É como um "número de apartamento" dentro daquele computador. Ela serve para achar o programa certo dentro do computador (tipo, o programa de site ou o programa de e-mail).
Questão 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

Resposta: É muito importante! Se um servidor web não consegue usar HTTPS, significa que ele não consegue fazer a conexão segura e as informações não serão protegidas. Isso faz com que o site seja considerado "não seguro", afastando as pessoas e podendo até não funcionar direito com coisas mais novas da internet.
Questão 19: Cite dois exemplos de servidores web e explique suas principais características.

Resposta: 1. Apache: É um dos mais antigos e populares. É bem flexível e dá para configurar de muitas formas. 2. NGINX: É conhecido por ser muito rápido e bom para lidar com muitos acessos ao mesmo tempo. É usado em sites grandes e movimentados.
Questão 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.

Resposta: Hospedagem compartilhada é quando seu site fica em um servidor que é dividido com vários outros sites, como se todos morassem na mesma casa e compartilhassem a cozinha e o banheiro. Ela é usada para sites pequenos, blogs simples ou para quem está começando e quer gastar pouco, pois é a opção mais barata e fácil de usar.
Questão 21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.

Resposta: A função de um servidor de aplicação é "rodar" os programas do site, fazendo as coisas mais inteligentes acontecerem, como buscar informações num banco de dados e mostrar um resultado personalizado. Já os servidores que só servem arquivos estáticos apenas entregam coisas prontas, como um documento PDF ou uma foto, sem fazer nenhum cálculo ou mudar as coisas na hora.
Questão 22: Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.

Resposta: Um servidor usado para projetos em Python é o Gunicorn. Ele é como um "gerente de tarefas" para seu programa Python. Quando chega um pedido, o Gunicorn pega esse pedido e dá para uma das suas "equipes" (chamadas workers) que rodam seu programa Python. Depois que o programa termina de fazer o que precisa, o Gunicorn pega o resultado e manda de volta.
Questão 23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?

Resposta: O WildFly (antigo JBoss) é um servidor grande e completo feito para programas em Java (os programas mais complexos de Java). Ele é indicado para projetos maiores e de empresas que usam muitas ferramentas e tecnologias específicas do Java.
Questão 24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?

Resposta: A porta 21 é usada para o FTP (File Transfer Protocol). Ela serve para "conversar" com um servidor para enviar e receber arquivos, tipo quando você usa um programa para colocar as fotos do seu site no lugar certo. É por essa porta que você dá comandos como "me mostra os arquivos" ou "envia este arquivo".
Questão 25: Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

Resposta: O LiteSpeed é compatível com o Apache porque ele consegue entender e usar os mesmos arquivos de configuração que o Apache usa. As vantagens disso são que é muito mais fácil trocar do Apache para o LiteSpeed sem ter que mudar tudo, e mesmo assim ter um site mais rápido e que usa menos recursos.
Questão 26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?

Resposta: O Microsoft IIS é um programa da Microsoft que serve para rodar sites e sistemas, principalmente em computadores que usam o sistema Windows. Ele é usado geralmente quando os programas foram feitos com tecnologias da Microsoft, como .NET (C#), ou em empresas que já usam muitos outros sistemas da Microsoft.
Questão 27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.

Resposta: O Node.js é como uma "ferramenta" que permite que você use a linguagem JavaScript (que normalmente é de site) para fazer programas que rodam no servidor, por trás do site. Ele é muito usado para: Fazer "serviços" que outros programas usam (chamados APIs). Aplicativos de bate-papo (chat) e jogos online. Sites que se atualizam sozinhos, sem precisar recarregar a página.
Questão 28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.

Resposta: A função do GlassFish é ser um ambiente para rodar programas grandes e complexos feitos em Java (os programas chamados Java EE). Ele é destinado a programas feitos na linguagem Java.
Questão 29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?

Resposta: Puma e Unicorn são programas que ajudam a rodar sites feitos na linguagem Ruby (especialmente com o Ruby on Rails). Eles são usados em projetos que utilizam essa linguagem para que o site funcione bem quando muitas pessoas acessam.
Questão 30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?

Resposta: Escolher o tipo certo de hospedagem é muito importante porque isso afeta diretamente se o seu site vai ser rápido ou lento, se ele vai aguentar muitas visitas ou vai cair, se vai ser seguro ou não, e quanto você vai gastar. Se você escolher errado, pode ter problemas com o site ou ele não conseguir crescer.
